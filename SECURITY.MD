# SECURITY REPORT

## 1. SQL Injection
Attack: Malicious DB queries
Fix: Use prepared statements

## 2. XSS
Attack: Script injection
Fix: Remove HTML tags

## 3. Prompt Injection
Attack: AI manipulation
Fix: Filter dangerous input

## 4. Rate Limiting Attack
Attack: API flooding
Fix: Limit requests

## 5. Unauthorized Access
Attack: No authentication
Fix: JWT protection

- SQL Injection → Data theft risk
- XSS → Browser attack
